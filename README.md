# Агентна система технічної підтримки

Інтелектуальна система автоматичної обробки запитів клієнтської підтримки, побудована на базі [LangGraph](https://github.com/langchain-ai/langgraph) та GPT-4o. Система класифікує запити користувачів, шукає відповіді у базі знань і формує відповіді, а за потреби — ескалює звернення до оператора.

## Можливості

- **Класифікація запитів** — автоматично визначає тип (проблема / питання / скарга), категорію (пароль / оплата / технічна / акаунт / загальне) та терміновість (низька / середня / висока)
- **Семантичний пошук** — знаходить релевантні відповіді у базі FAQ за допомогою FAISS-векторного сховища
- **Генерація відповідей** — формує відповідь на основі знайдених матеріалів за допомогою GPT-4o
- **Ескалація** — автоматично передає до оператора запити з високою терміновістю, скарги та юридичні питання

## Архітектура

Граф агента виконує такі кроки:

```
START → classify_query → search_faq → draft_response → check_escalation → END
```

### Структура проєкту

```
technical_support_agentic_system/
├── app.ipynb                    # Основний Jupyter-ноутбук
├── lib/
│   ├── classification.py        # TypedDict-схеми для стану агента
│   ├── llm.py                   # Фабрика OpenAI-клієнта
│   ├── nodes.py                 # Вузли графа агента
│   └── utils.py                 # Утиліти бази знань (FAISS)
├── knowledge_base/
│   └── faq_database.json        # База FAQ
└── pyproject.toml
```

## Вимоги

- Python >= 3.12
- [uv](https://docs.astral.sh/uv/) — менеджер пакетів
- Ключ API OpenAI

## Запуск через uv

### 1. Встановлення залежностей

```bash
uv sync
```

### 2. Встановлення ключа API OpenAI

```bash
export OPENAI_API_KEY="sk-..."
```

### 3. Запуск Jupyter-ноутбука

```bash
uv run jupyter notebook app.ipynb
```

Або запустити JupyterLab:

```bash
uv run jupyter lab
```

Після відкриття ноутбука виконайте всі комірки послідовно — система побудує граф агента, завантажить базу знань і запустить тестові запити.

## Метрики

Під час тестування на 5 запитах:

| Метрика | Результат |
|---|---|
| Завершеність задач | 100% (5/5) |
| Середня релевантність відповіді | 100% |
| Рівень ескалації | 40% (2/5) |
